# 05-02 Plan: Security Mode and Full Backup UX

## Goal

把 `keychain/passphrase` 模式从后端扩展到 typed facade 与 UI，对 full backup 的导入导出形成完整交互闭环。

## Deliverables

- `backupSecurityMode` 持久化到 `preferences.json`，默认解析为 `keychain`。
- keychain adapter 通过 `zalando/go-keyring` 读取/创建 secret。
- full export：先选路径；`passphrase` 模式下要求两次输入一致；`keychain` 模式直接执行。
- full import：先选路径；若返回 `backup.passphrase_required`，在同一路径上打开单次 passphrase 重试对话框。

## Verification

- frontend bridge/service 单测覆盖 backup/settings unwrap
- full export mismatch 阻断调用
- full import 在 `passphrase_required` 后可重试同一路径
- `wails build -clean`
